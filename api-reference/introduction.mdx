## Overview

The Audimee API lets you list available AI voice models and submit audio files for voice conversion. Conversions run asynchronously — you create a conversion job, then poll for its status until it's done.

## Base URL

```
https://audimee.com/api/v1
```

## Authentication

All endpoints require a bearer token in the `Authorization` header:

```
Authorization: Bearer {token}
```

## Quick example

```bash
# 1. Pick a voice model
curl https://audimee.com/api/v1/voice-models \
  -H "Authorization: Bearer {token}"

# 2. Start a conversion
curl -X POST https://audimee.com/api/v1/conversions \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "voiceModelId": "101",
    "inputFileUrl": "https://example.com/audio/my-recording.wav",
    "conversionStrength": 0.7
  }'

# 3. Poll until done
curl https://audimee.com/api/v1/conversions/{id} \
  -H "Authorization: Bearer {token}"
```
